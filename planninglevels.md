### מערכת ניקוד והצלחה
- **3 כוכבים**: כל המשימות הושלמו + נשארו 5+ דקות → 50 מטבעות
- **2 כוכבים**: כל המשימות הושלמו + נשארו 0-5 דקות → 30 מטבעות
- **1 כוכב**: חסרה משימה אחת לא קריטית או איחור של 1-3 דקות → 10 מטבעות
- **0 כוכבים**: חסרו 2+ משימות או איחור של 3+ דקות → נסה שוב

### משימות קריטיות (חובה) vs. משימות רצויות
**קריטיות** (אם חסרות = מקסימום 1 כוכב):
- צחצוח שיניים
- לבישה
- הכנת תיק בית ספר
- נעילת נעליים

**רצויות** (אם חסרות = עדיין אפשר 2-3 כוכבים):
- קימה ממיטה (אפשר לקום לאט)
- שירותים (אפשר בבית ספר)
- מקלחת (אפשר בערב)
- ארוחת בוקר (רצוי אבל לא הכרחי)

### מטרה פדגוגית
תרגול עצמאי של שגרת בוקר בסיסית עם ניהול זמן פשוט. הילד לומד לתעדף משימות ולעבוד בקצב נוח.

### שלב 2.1 - "הבוקר הראשון שלי"

**מסך פתיחה:**
```
┌──────────────────────────────────────┐
│   🌅 בוקר ראשון בבית הספר! 🌅       │
│                                      │
│   רמה: קלה ⭐                        │
│   זמן זמין: 45 דקות                 │
│                                      │
│   משימות:                            │
│   • קימה מהמיטה                     │
│   • שירותים                          │
│   • צחצוח שיניים                    │
│   • לבישה                            │
│   • ארוחת בוקר                      │
│   • הכנת תיק                         │
│                                      │
│   💡 קח את הזמן - אין לחץ!          │
│                                      │
│   [התחל]                             │
└──────────────────────────────────────┘
```

**פירוט משימות:**

#### 1. קימה מהמיטה
- **זמן**: 1 דקה
- **מיקום**: חדר שינה
- **הנחיה**: "בוקר טוב! לחץ על הדמות כדי לקום"
- **תנאי הצלחה**: לחיצה על הדמות

#### 2. שירותים
- **זמן**: 2 דקות
- **מיקום**: חדר אמבטיה
- **הנחיה**: "לך לשירותים"
- **אנימציה**: הדמות נכנסת ויוצאת מהשירותים
- **תנאי הצלחה**: השלמת האנימציה

#### 3. צחצוח שיניים
- **זמן**: 2 דקות
- **מיקום**: חדר אמבטיה (כיור)
- **הנחיה**: "צחצח שיניים - חשוב!"
- **תנאי הצלחה**: לחיצה על מברשת השיניים
- **בונוס**: +10% אנרגיה, "חיוך נקי!"

#### 4. לבישה
- **זמן**: 3 דקות
- **מיקום**: חדר שינה (ארון)
- **הנחיה**: "בחר בגדים מתאימים לבית ספר"
- **אינטראקציה**: מסך בחירת בגדים (כמו ב-Tutorial)
- **תנאי הצלחה**: בחירת חולצה + מכנסיים + גרביים

#### 5. ארוחת בוקר
- **זמן**: 5 דקות
- **מיקום**: מטבח
- **הנחיה**: "אכול משהו - זה יעזור לך להיות ערני!"
- **אופציות**:
  - דגני בוקר (5 דק') → +30% אנרגיה
  - טוסט (3 דק') → +20% אנרגיה
  - קרואסון (2 דק') → +10% אנרגיה
  - דילוג (0 דק') → -20% אנרגיה
- **תנאי הצלחה**: בחירה באחת מהאופציות

#### 6. הכנת תיק
- **זמן**: 5 דקות
- **מיקום**: חדר שינה (שולחן)
- **הנחיה**: "אסוף את הדברים לבית ספר"
- **מיני-משחק**: מצא ולחץ על:
  - 📕 ספר (על המדף) - 1 דקה
  - 📓 מחברת (על השולחן) - 1 דקה
  - ✏️ קלמר (במגירה) - 1 דקה
  - 🍎 חטיף (במטבח) - 2 דקות (אופציונלי)
- **תנאי הצלחה**: איסוף של לפחות 3 פריטים חובה

**זמן כולל משוער**: 18 דקות
