# MorningRush

**"התארגן, תכנן וצא בזמן – משחק ניהול זמן שמלמד ילדים שגרת בוקר נכונה"**

---

##  על המשחק

**Morning Rush** 
הוא משחק ניהול זמן חווייתי שמטרתו ללמד ילדים (גילאי 5-12) לנהל את שגרת הבוקר שלהם בצורה עצמאית ויעילה.

המשחק משחזר את חוויית הבוקר בצורה פשוטה, משחקית ומעודדת – תוך חיזוק תפקודים ניהוליים כמו תכנון, סדרי עדיפויות, ניהול זמן וגמישות מחשבתית.

בכל שלב הילד מתמודד עם משימות בוקר שונות: קימה, היגיינה, לבישה, ארוחת בוקר והכנת תיק – תוך התמודדות עם הפרעות ובעיות בלתי צפויות.

השחקן מקבל משוב מיידי, צובר כוכבים ומטבעות, ויכול לשפר ביצועים ולשדרג את הדמות והבית לאורך הדרך.

המשחק נועד לשלב חוויה של למידה, הנאה ופיתוח מיומנויות חיים.

##  על הפרויקט

הפרויקט מפותח במסגרת קורס פיתוח משחקי מחשב, בשיתוף פעולה בין החוג למדעי המחשב לבין החוג לריפוי בעיסוק.

מטרתו היא לתרום לפיתוח מיומנויות חיים ועצמאות בילדים באמצעות משחקים אינטראקטיביים, חווייתיים וחינוכיים.

המשחק משלב עקרונות עיצוב משחקים, פסיכולוגיה התפתחותית וריפוי בעיסוק ליצירת כלי למידה יעיל ומהנה.
